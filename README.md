# 🦋 Squish Squad

A Y2K-hyperpop bilingual (English / Arabic) squishy shop for Dubai 🇦🇪✨

**Live site:** _add Vercel URL here after deploy_

## ✨ Features

- 🌈 **Y2K hyperpop aesthetic** — holographic gradients, chrome, butterflies, sparkles
- 🇦🇪 **Bilingual** — English + Arabic with full RTL support and one-click language toggle
- 💰 **AED prices** with free shipping over AED 100
- 🛒 **Themed slide-out cart** with quantity controls, free-ship hint, and live total
- 🚚 **Multi-step checkout** — delivery form for UAE (name, phone, emirate, address, notes)
- 🎉 **Order confirmation** screen with order number and delivery summary
- 📸 **Photo upload** for products (auto-resized client-side)
- ⚙️ **Admin panel** — full CRUD on products: add / edit / delete / reset
- 🎁 **Mystery Box** with rarity reveals (Common / Rare / Epic / Legendary)
- 💖 100% client-side — no backend needed. Data lives in localStorage.

## 🛠️ Tech

Pure HTML / CSS / vanilla JavaScript — no build step.

| File | Purpose |
| --- | --- |
| `index.html` | Home page with featured products |
| `shop.html` | Full product grid with category filters |
| `mystery.html` | Mystery Box reveal + pricing tiers |
| `admin.html` | Control panel for managing products |
| `styles.css` | All Y2K + RTL styling |
| `script.js` | Translations, cart, checkout, admin CRUD, photo upload |
| `vercel.json` | Deployment config (clean URLs + cache headers) |
